# 習慣Todo - プライバシーポリシー

このリポジトリは、iOSアプリ「習慣Todo」のプライバシーポリシーを公開するためのものです。

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubにプッシュ
2. リポジトリの Settings > Pages に移動
3. Source を `main` ブランチの `/ (root)` に設定
4. Save をクリック

数分後、以下のURLでアクセス可能になります：
`https://[your-username].github.io/todohabit_doc/`

## ファイル構成

- `index.html` - プライバシーポリシーのHTMLページ
- `README.md` - このファイル

## カスタマイズ

公開前に、`index.html`の以下の部分を編集してください：

1. **お問い合わせメールアドレス**（10. お問い合わせ）
   ```html
   <a href="mailto:your-email@example.com">your-email@example.com</a>
   ```
   → 実際のメールアドレスに変更

2. **最終更新日**（必要に応じて）
   ```html
   <p class="last-updated">最終更新日: 2026年2月3日</p>
   ```
